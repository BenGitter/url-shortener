URL Shortener service

This API will change a provided url to an a simple anonymous url. You can then open that url and you will be redirected to the orignal url.

NOTE: Provided url should look like: http://www.example.com

Example creation usage

https://honey-diver.hyperdev.space/new/http://www.google.com

Example creation output

{ "original_url": "http://google.com", "short_url": "https://honey-diver.hyperdev.space/57ceae9e290fb2099cf6e619"  }

Usage

https://honey-diver.hyperdev.space/57ceae9e290fb2099cf6e619

Will redirect to:

http://www.google.com
